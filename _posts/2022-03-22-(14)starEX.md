# 별 찍기 (연습 문제)

- 최초 작성일: 2021년 3월 21일(월)

## 내용

```c#
using System;

namespace ex2
{
    class Program
    {
        static void Main(string[] args)
        {
            // 별 찍기
            for (int i = 0; i < 5; i++) 
            {
                for (int j = 0; j <= i; j++) 
                {
                    Console.Write("*");
                }
                Console.WriteLine();
            }
        }
    }
}
```
